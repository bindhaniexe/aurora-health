## 🌟 What is Aurora?

Aurora is a **mobile-first AI health companion** that makes health tracking feel natural and motivating. Instead of manually logging data into forms, you simply **talk to Aurora** — it understands your context, logs the data for you, and gives you personalized daily insights.

### Core Experience

- 🎙️ **Voice-first interaction** — tell Aurora "I drank 500ml of water" and it logs it
- 💧 **Hydration tracking** with an animated virtual water bottle
- 😴 **Sleep tracking** and weekly trend analysis
- ✅ **Habit creation, streaks, and completion** tracking
- 🧠 **Daily AI insight cards** generated by Groq for personalized motivation
- 🎨 **Beautiful, vibrant UI** with purple-pink gradient accents and smooth animations

## 🤖 AI Features

### Voice Companion (OpenAI Realtime API)

The voice companion is the heart of Aurora. Tap the mic and speak naturally:

> *"I drank 2 glasses of water"* → logs 500ml  
> *"I slept for 7.5 hours last night"* → logs sleep  
> *"Add a new habit: morning stretch"* → creates the habit  
> *"How am I doing today?"* → AI summarizes your progress

**Agent Tools the AI can call:**

| Tool | Description |
|------|-------------|
| `log_water(amount_ml)` | Logs a hydration entry |
| `log_sleep(hours, quality?)` | Logs a sleep session |
| `create_habit(name, frequency)` | Creates a new habit |
| `complete_habit(habit_name)` | Marks a habit complete for today |
| `get_health_summary()` | Fetches today's data for context |

**Security:** The OpenAI API key never lives in the client. A Supabase Edge Function (`realtime-session`) creates an ephemeral session token and returns it to the app.

### Daily Insights (Groq — llama-3.3-70b)

Each morning, Aurora generates a personalized, single-sentence motivational insight based on your health data. Results are cached daily in Supabase so the API is only called once per day.

## 🗄️ Database Schema

Managed via Supabase migrations:

| Table | Purpose |
|-------|---------|
| `profiles` | User profile, goals, preferences, memory notes |
| `hydration_logs` | Per-drink water intake entries |
| `sleep_logs` | Nightly sleep sessions with quality rating |
| `habits` | User-defined habits with frequency config |
| `habit_completions` | Daily completion records (streak logic) |
| `insights_cache` | Cached daily Groq insight cards |

All tables have **Row Level Security** — users can only access their own data.

## 🚀 Getting Started

### Prerequisites

- Node.js 18+
- [Expo CLI](https://docs.expo.dev/get-started/installation/)
- [Expo Go](https://expo.dev/go) app on your phone, **or** an Android/iOS emulator
- A Supabase project (free tier works)
- OpenAI API key (for voice companion)
- Groq API key (for daily insights)

### 1. Clone & Install

```bash
git clone https://github.com/YOUR_USERNAME/aurora.git
cd aurora
npm install
```

### 2. Configure Environment

Create a `.env.local` file in the project root:

```env
EXPO_PUBLIC_SUPABASE_URL=https://your-project.supabase.co
EXPO_PUBLIC_SUPABASE_ANON_KEY=your-supabase-anon-key
EXPO_PUBLIC_GROQ_API_KEY=your-groq-api-key
```

> ⚠️ **Never commit your `.env.local` file.** It is already in `.gitignore`.

### 3. Set Up the Database

Push the migrations to your Supabase project:

```bash
npx supabase db push
```

Or run them manually in the Supabase SQL Editor from `supabase/migrations/`.

### 4. Deploy the Edge Function

```bash
npx supabase functions deploy realtime-session
```

Set your OpenAI key as a Supabase secret:

```bash
npx supabase secrets set OPENAI_API_KEY=your-openai-api-key
```

### 5. Start the App

```bash
npm start          # Expo Go (scan QR code)
npm run android    # Android emulator / device
npm run ios        # iOS simulator / device
```
